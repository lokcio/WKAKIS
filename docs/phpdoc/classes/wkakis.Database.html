completeQuery()

completeQuery(array<mixed,string>  $options, array<mixed,string>  $queryCompletions) : string

Tworzy dopełnienie zapytania SQL poprzez warunki przekazane w tablicy. Pozwalana na zachowanie poprawnej kolejnosci warunków oraz filtruje według tablicy $queryCompletions.

$queryCompletions = array(
    'where'=> ' WHERE ',
    'group'=> ' GROUP BY ',
    'having'=> ' HAVING ',
    'order'=> ' ORDER BY ',
    'limit'=> ' LIMIT '
);

$options = array(
    'select'=> '*',
    'from'=> 'table',
    'where'=> 'column=5',
    'group'=> 'column2',
    'order'=> 'id DESC',
    'limit'=> 21
);

$db = new Database($dbconfig);

echo $db->prepareAndExecute($options, $queryCompletions);
// 'WHERE column=5 GROUP BY column2 ORDER BY id desc LIMIT 21'

Parameters

array<mixed,string> $options

Opcje zapytania SQL z możliwymi dopełnieniami.
array<mixed,string> $queryCompletions

Asocjacyjna tablica dopełnień.

Returns

string —

Dopełnienie do zapytania SQL.
